# TR JN3C — Site Institucional

Landing page institucional da **TR JN3C LTDA**, transportadora especializada em contêineres e carga seca, sediada em Itajaí/SC.

## Estrutura

```
transportadora-trjn3c/
├── index.html                          # Página principal (SEO otimizado)
├── assets/
│   ├── images/
│   │   ├── logo-transportadora-trjn3c.png   # Logo principal (fundo transparente)
│   │   └── og-image.jpg                     # Imagem para compartilhamento (1200x630)
│   └── favicon/
│       ├── favicon.ico                      # Ícone do site (multi-resolução)
│       └── favicon-32x32.png                # Ícone 32px
└── README.md
```

## Tecnologias

- HTML5 semântico puro (sem frameworks)
- CSS inline com variáveis customizadas
- JavaScript vanilla (scroll animations)
- Schema.org para SEO local (LocalBusiness)
- Open Graph para compartilhamento em redes sociais

## Deploy

Hospedado via **GitHub Pages**.

Para publicar:
```bash
git add .
git commit -m "deploy: atualiza site"
git push origin main
```

O site fica disponível em:
`https://seuusuario.github.io/transportadora-trjn3c`

Com domínio próprio: aponte o DNS para `seuusuario.github.io` e configure em *Settings → Pages → Custom domain*.
